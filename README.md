## ğŸ¯ What is QC-Kit?

QC-Kit is a **Cognitive Squad** of 15 Quality Control Specialists designed for the **Antigravity Native Protocol (ANP)**.

It replaces the "Single Chatbot" paradigm with a **Team of 15 QC Experts**:
*   Need Test Cases? Summon **`@qc-testcase`**.
*   Need Automation Scripts? Summon **`@qc-automation`**.
*   Need Security Audit? Summon **`@qc-security`**.

Each specialist is equipped with **System 2 Thinking** (Reflective Loops) to "Stop & Think" before answering, ensuring **High-Assurance** test coverage.

## ğŸ§  System 2 Intelligence

All agents follow a **Reflective Cognitive Loop**:
1.  **Analysis (System 1)**: Fast pattern matching.
2.  **Action (System 1)**: Draft test cases/scripts.
3.  **Reflection (System 2)**: **STOP & THINK**.
    *   *Critic*: "Did I cover edge cases? Boundary values?"
    *   *Action*: Generate additional cases.
4.  **Output**: Verified, Complete Test Artifacts.
